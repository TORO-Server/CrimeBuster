# CrimeBuster

Gemma 4 AI モデルを基盤とした Minecraft サーバー用犯罪抑止システム。

プレイヤーのチャットログ、会話の文脈、およびプレイヤー間の相関データをリアルタイムで解析し、犯罪係数（Crime Score）を算出する。

算出された係数に基づき、警告、Kick、または Ban を自動的に執行する。

本システムは AI チャット解析、高度な文脈推定、および自動処罰執行の三要素で構成される。

AI 解析では単一の発言だけでなく、直近のチャットログとプレイヤー間の親密度（Intimacy）をパラメータとして注入することで、ジョークや親愛の情に伴う「いじり」の誤検知を最小限に抑制する。

犯罪係数が規定の閾値（Kick: 70, Ban: 100）を超過した場合、システムは即座に処罰プロセスを開始する。

## コマンド一覧

利用可能なコマンドは以下の通りである。

/crime: 実行者自身の犯罪係数を確認する。

/crime info <player>: 対象プレイヤーの犯罪係数を確認する（要権限）。

/crime set <player> <value>: 犯罪係数を任意の値に変更する（要権限）。

/crime reset <player>: 犯罪係数を 0 にリセットする（要権限）。

/crime reload: 設定ファイル（config.yml）およびプロンプト定義（prompt.md）を再読み込みする（要権限）。

## 設定およびデータ管理

プラグインフォルダ内の config.yml にて API 端点および処罰閾値を定義する。

また、prompt.md により AI への判定アルゴリズムを外部定義することが可能である。

プレイヤーごとの犯罪データ、親密度、および交流履歴は JSON 形式で永続化されており、
プレイヤーの再参加時にも継続的な監視が可能である。
